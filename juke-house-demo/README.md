# Juke House

Juke House is a Version 1 demo for a video upload and tipping platform for approved entertainers and artists.

## What is included

- Home page
- Artist application page
- Browse artists page
- Artist profile page
- Video feed page
- Video watch page
- Fan subscription page
- Gift/tip page
- Artist dashboard
- Admin dashboard

The demo includes local browser storage for accounts, applications, follows, gifts, subscriptions, upload drafts, category controls, and admin approvals.

## Open locally

Open `index.html` in a browser.

## Publish with GitHub Pages

1. Create a new GitHub repository.
2. Upload the files in this folder.
3. Go to the repository Settings.
4. Open Pages.
5. Set Source to `Deploy from a branch`.
6. Choose the `main` branch and `/root`.
7. Save.

GitHub will give you a public website link after it finishes publishing.
